# <a name="create-android-applications-with-system-center-configuration-manager"></a>使用 System Center Configuration Manager 创建 Android 应用程序

*适用范围：System Center Configuration Manager (Current Branch)*

除了创建应用程序的其他 System Center Configuration Manager 要求和过程，在创建和部署适用于 Android 设备的应用程序时还必须考虑以下注意事项。  

## <a name="general-considerations"></a>一般注意事项

Configuration Manager 支持部署以下 Android 应用类型：

|设备类型|受支持的文件|
|-|-|
|Android|.apk|

支持以下部署操作：

|设备类型|支持的操作|
|-|-|
|Android|“可用”、“必需”（但用户必须同意安装）、“卸载”（同样需要许可）。|
