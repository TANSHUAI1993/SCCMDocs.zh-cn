## <a name="bkmk_repair"></a> 对修复应用程序的改进
<!--1357866-->

软件中心现提供了一个新按钮，可用于修复  应用程序。 当为应用程序配置修复程序时，用户可以从软件中心启动命令。 

有关详细信息，请参阅[修复应用程序](/sccm/core/get-started/capabilities-in-technical-preview-1807#bkmk_app-repair)。
