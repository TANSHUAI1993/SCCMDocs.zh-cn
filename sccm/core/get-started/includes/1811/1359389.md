## <a name="bkmk_posh"></a> 对运行 PowerShell 脚本任务序列步骤的改进
<!--1359389--> 现在，可以在运行 PowerShell 脚本步骤中直接输入 Windows PowerShell 代码。 此更改允许你在任务序列期间运行 PowerShell 命令，而无需先使用脚本创建和分发包。
