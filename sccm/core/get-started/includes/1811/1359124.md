## <a name="bkmk_opmdm"></a> 本地 MDM 不再需要 Intune 连接
<!--1359124--> 不再需要配置 Microsoft Intune 订阅的本地 MDM 先决条件。 组织仍需要 Intune 许可证才能使用此功能。 

现在可以使用 Configuration Manager 来管理 Surface Hub 设备，而无需 Intune 连接。
