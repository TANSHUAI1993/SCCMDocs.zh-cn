## <a name="bkmk_posh"></a> 对运行 PowerShell 脚本任务序列步骤的改进
<!--3556028-->根据反馈，运行 PowerShell 脚本任务序列步骤现在包括以下改进：  

- 使用“作为以下帐户运行此步骤”选项指定备用用户凭据。  

- 指定“超时(分钟)”。 如果该脚本运行时间超出超时值，此选项将结束该步骤。  

- 通过“开始于”字段指定工作目录。  

- 在“选项”选项卡上，指定“成功代码”。 添加该脚本中的其他退出代码，该步骤应评估为成功。
