## <a name="bkmk_o365lang"></a> Office 365 更新的额外的语言
<!--3555955-->

Configuration Manager 现在支持 Office 365 客户端更新支持的所有语言。 现在，更新工作流将“Windows 更新”的 38 种语言与“Office 365 客户端更新”的 103 种语言分开。 

若要选择必要的语言，请使用以下位置的“语言选择”页面：
- 创建自动部署规则向导
- 部署软件更新向导
- 下载软件更新向导
- 自动部署规则属性
